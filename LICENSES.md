# Licencias
 ## PostgreSQL
 PostgreSQL se distribuye bajo la Licencia PostgreSQL (una licencia permisiva de tipo open source similar a la BSD o MIT).
https://www.postgresql.org/

## OpenSSH
OPenSSh se distribuye libremente con una licencia BSD.Aunque todo el codigo fuente de SSH está disponible , existen reestricciones al respecto.
http://openssh.org/

## Guacamole
Es desarrollado por una comunidad de colaboradores como proyecto de la Apache Software Foundation , tiene licencia Apache.
https://guacamole.apache.org/
